Es necesario mantener una narrativa coherente, mostrando el porqué y el cómo de cada acción para evitar malentendidos o conflictos en la historia. La mejor manera de garantizar una buena dinámica es acordar este tipo de situaciones entre los participantes antes de llevarlas a cabo.

Tener en cuenta, que si hay recurrencia en el abandono de temas, se procederá a sancionar al usuario en cuestión.                                        
								</li>

									                                        Este foro se construye entre todos. Se valora la disposición a interactuar, planificar tramas, sugerir nuevas ideas. No se obliga a nadie a forzar vínculos, pero no queremos que existan islas de rol donde dos personajes roleen entre sí y pasen del resto del foro. El ambiente, la inspiración y la perdurabilidad del foro depende de la colaboración grupal.
